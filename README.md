# Solar Collector Calculator

This calculator calculates the solar collector's daily energy output (measured in hours of perpendicular sunlight) based on roof angle and orientation. Part of the quest [Bless Me, Father, for I Have Sin(n)ed](https://scinquest.com/@gershep/q/bless-me-father-for-i-have-sinned) and used to settle an argument between my father and me.
